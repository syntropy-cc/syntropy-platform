# Tutorials

Tutorials are learning-oriented: they take you through a complete workflow from start to finish so you can see how the platform works and achieve something meaningful.

| Tutorial | What you'll do | Time |
|----------|----------------|------|
| [01 — Onboarding to First Artifact](01-onboarding-first-artifact.md) | Sign up, get oriented, and publish your first artifact from a Learn fragment | ~20 min |
| [02 — Complete a Learn Track](02-complete-learn-track.md) | Work through a track: fragments, theory, and artifacts until you have a complete project | ~1–2 hours |
| [03 — Create Institution and Project](03-create-institution-and-project.md) | Create a digital institution with governance and your first project and issue | ~25 min |
| [04 — Contribute to a Hub Project](04-contribute-to-hub-project.md) | Find an issue, work in the IDE, and submit a contribution | ~30 min |
| [05 — Publish a Labs Article](05-publish-labs-article.md) | Create a laboratory, write an article, publish it, and receive reviews | ~40 min |

Start with [Onboarding to First Artifact](01-onboarding-first-artifact.md) if you are new; then pick the tutorial that matches your goal (Learn, Hub, or Labs).
