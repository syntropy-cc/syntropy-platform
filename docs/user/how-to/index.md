# How-to Guides

How-to guides are task-oriented: they give you direct steps to accomplish a specific goal. Use them when you already know the basics and need a recipe.

| Guide | Goal |
|-------|------|
| [Authenticate with the API](authenticate-api.md) | Get a Bearer token and call protected endpoints |
| [Create an Artifact](create-artifact.md) | Create, submit, and publish a DIP artifact via API |
| [Create an Institution](create-institution.md) | Create a digital institution with a governance template |
| [Submit a Contribution](submit-contribution.md) | Submit an artifact as a contribution to a Hub issue |
| [Submit a Labs Review](submit-review-labs.md) | Submit a peer review for a published Labs article |
| [Use an IDE Session](use-ide-session.md) | Create and use an IDE session (and WebSocket) via API |
| [Sponsor a Creator](sponsor-creator.md) | Support a creator with a one-time or recurring sponsorship |
